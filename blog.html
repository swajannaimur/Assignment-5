Question-1: What are the different ways to select an element in the DOM?

Question-2: What is the difference between innerHTML, innerText, and textContent ?

textContent shows the raw text even the hidden also. innerText shows only the visible text in an element. and innerHTML shows which HTML is written for an element.

Question-3: What is event delegation in the DOM?

Question-4: What is event bubbling in the DOM?

Event first hits the innermost element, then it goes it's parent element. this loop goes on till the event hit outermost target element. this loop is called Event Bubbling
